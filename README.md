# Yiting Zhao Online CV

Static online CV website for GitHub Pages.

Recommended public URL:

```text
https://Yiting-Z-hao.github.io
```

## Publish With GitHub Pages

1. Create a public GitHub repository named `Yiting-Z-hao.github.io`.
2. Upload `index.html`, `styles.css`, the `assets` folder, and this `README.md`.
3. GitHub Pages will publish the site at `https://Yiting-Z-hao.github.io`.

If the site does not appear immediately, open the repository settings, go to Pages, and confirm the source is set to the `main` branch.
